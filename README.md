# Product Preview Card Component

A responsive product preview card built using **HTML and CSS** as part of a Frontend Mentor challenge.  
The project focuses on creating a clean and modern UI component that adapts to different screen sizes.

## Overview

This project recreates a product preview card for a perfume product. The design includes a product image, description, pricing information, and an interactive "Add to Cart" button.

The layout changes depending on the screen size:
- **Mobile:** vertical card layout
- **Desktop:** horizontal card layout

This helps practice **responsive design and modern CSS layout techniques.**

## Features

- Responsive design for mobile and desktop
- Product image with optimized positioning using `object-fit` and `object-position`
- Clean typography using **Google Fonts**
- Interactive hover state for the button
- Flexbox-based layout structure

## Built With

- HTML5
- CSS3
- Flexbox
- Responsive design with media queries
- Google Fonts

## What I Learned

While building this project, I practiced:
- structuring semantic HTML
- building responsive layouts using Flexbox
- controlling image cropping with `object-fit` and `object-position`
- creating interactive button hover effects
